## 하모니카는 어떤 프로젝트 인가요?

한국어를 사용하는 사용자가 좋은 오픈소스 소프트웨어를 찾고 한국어 번역에 참여하고 발전시켜서 자유 라이선스에 따라서 이를 공개하도록 허용하고 권장하는 것입니다.
이렇게 오픈소스 소프트웨어를 효율적으로 한국어 접근성을 높여 보급하는 것입니다.
